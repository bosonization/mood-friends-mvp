# Mood Orbit / Emotion Dock update

## Changes

- Mood Orbit bubble center is now the friend's profile icon.
- Mood appears as a small badge, not as text inside the bubble.
- Bubble still shows name and time.
- Bubble text size changes by hot/fresh/normal bubble size.
- Time in small bubbles is compact, such as `16h` or `2d`; full time remains in the selected detail panel.
- Temperature expression uses rings and glow:
  - 1 hour: red, large, glowing
  - 6 hours: green, medium, glowing
  - older: quiet translucent bubble
- Mobile bottom navigation is replaced by Emotion Dock.

## Supabase

No SQL is required for this update.

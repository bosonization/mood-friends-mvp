# Contact actions update

## 変更内容

- Mood Orbit詳細カードの「1時間以内 / 6時間以内 / その他」の凡例を削除
- 詳細カードに「外部で連絡する」パネルを追加
- 追加した導線
  - 共有
  - LINE
  - SMS
  - メール
  - 電話（相手名コピー + 端末の連絡先から電話する前提）
  - 文面コピー

## 設計方針

- eMoodition内には電話番号、LINE ID、Instagram ID、メールアドレスを保存しない
- アプリ内チャット、DM、連絡先交換機能は持たせない
- InstagramはWebから特定DMを安定して開けないため、共有シート経由にする
- 連絡はすでに知っている相手に、外部アプリで行う前提

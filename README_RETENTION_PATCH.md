# eMoodition retention patch

## 変更内容

- 新規登録時にパスワード確認欄を追加
- パスワードと確認用パスワードが一致しない場合は登録を止めてエラー表示
- 初回プロフィール作成時の利用規約・プライバシーポリシーリンクを別タブ表示に変更
  - 入力中に規約を見てもフォーム内容がリセットされにくくなります
- Mood Orbitで友達が1〜4人のとき、最低5つのバブルが見えるように空バブルを追加
  - 友達0人のときは従来通り招待導線を表示
  - List表示には空バブルを出しません

## Supabase SQL

不要です。DB構造は変更していません。
